# Github Blog

![Github Blog](./public//logo.svg)

Essa aplicação utiliza a API do GitHub para buscar issues do repositório, dados do meu perfil para exibir com um blog.

- Listagem do perfil com imagem, número de seguidores, nome e outras informações disponíveis pela API do GitHub.
- Listar e filtrar todas as issues do repositório com um pequeno resumo do conteúdo dela
- Criar uma página para exibir um post (issue) completo

Apesar de serem poucas funcionalidades, você vai precisar relembrar conceitos como:

- Fetch / Axios
- Roteamento e React Router DOM
- Formulários
